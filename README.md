# RecordApp

A minimal web application to create and manage notes.

## 🚀 Tech Stack

- Frontend: Angular
- Backend: .NET Minimal API
- Database: SQLite

## ✨ Features (MVP)

- Create notes
- List notes
- Delete notes

## 📦 Project Structure

/backend → .NET API
/frontend → Angular app


## ⚙️ Getting Started

### 1. Clone the repository

git clone https://github.com/your-username/recordapp.git
cd recordapp

### 2. Run backend
cd backend
dotnet run

API will run on:
http://localhost:5000

### 3. Run frontend
cd frontend
npm install
ng serve

App will run on:

http://localhost:4200

## Configuration

Make sure the frontend API URL is set correctly in Angular environment files.
